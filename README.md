# mkhot-archive

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

mkhot-archive 是一个轻量级的静态页面归档仓库，主要用于存放和展示独立的 HTML 文档。本仓库不针对任何特定域名或网站，仅作为通用 HTML 页面的集合与存档。

## 目录结构

```
mkhot-archive/
├── pages/          # 存放各个独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：所有归档的 HTML 页面均放置在此目录下，每个页面独立存在，互不依赖。
- `assets/`：用于存放页面可能引用的公共资源，如样式表、脚本或图像文件。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档内容，可能涉及不同主题或用途。每个页面均为自包含的静态文件，可通过直接访问文件路径进行浏览。

- 页面名称通常反映其内容或归档时间。
- 页面内可能引用 `assets/` 目录下的资源文件，但建议尽量使用内联方式以减少外部依赖。
- 归档的页面在发布后原则上不会修改，以保持历史记录的完整性。

## 维护说明

- 本仓库由维护者不定期更新，新增的 HTML 页面会放入 `pages/` 目录。
- 欢迎提交 Issue 或 Pull Request 建议添加新的归档页面，但请确保内容合规且不侵犯第三方权益。
- 对于不再需要的页面，维护者可能会将其移入存档分支或直接删除。
- 仓库中不包含任何形式的广告、营销内容或诱导点击链接。
